# Tech-Services-Analytics-Engine
## Overview
Built an end-to-end ETL pipeline to clean, normalize, and load messy company data into a PostgreSQL database and created an interactive Power BI dashboard for market analysis.

## Tech Stack
- Python (Pandas)
- PostgreSQL
- SQLAlchemy
- Power BI

## Pipeline Flow
Raw CSV → Data Cleaning & Normalization → PostgreSQL → Power BI Dashboard

## Key Features
- Cleaned inconsistent team size and pricing formats
- Normalized data into relational tables
- Loaded data using SQLAlchemy
- Built interactive dashboard with filters for country, service, and awards

## Dashboard Insights
- Geographic distribution of companies
- Service-wise market share
- Pricing trends across services
- Award distribution analysis
